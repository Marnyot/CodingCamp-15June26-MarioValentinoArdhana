# Personal Dashboard

A client-side personal dashboard app with focus timer, task manager, quick links, and live greeting — all data stored locally in the browser.

## Features

- **Live Clock & Greeting** — Real-time clock, date, and time-based greeting with customizable name
- **Focus Timer** — Configurable Pomodoro countdown (1–999 min) with start/stop/reset
- **Task Manager** — Add, edit, complete, delete, and sort tasks (All / Active / Completed). Duplicate tasks are prevented
- **Quick Links** — Add and open URL shortcuts with automatic scheme normalisation
- **Dark Mode** — Theme toggle persisted across sessions

## Tech Stack

- HTML5 (semantic)
- CSS3 (custom properties, gradients, animations)
- Vanilla JavaScript (no frameworks)
- Browser Local Storage API (no backend)

## Usage

Open `index.html` in any modern browser. No server or build step required.
